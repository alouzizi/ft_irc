# ft_irc
Create your own IRC server in C++, fully compatible with an official client.  

## Installation & Run

Install using:

```bash
  git clone https://github.com/hixcoder/ft_irc.git && cd ft_irc
```

Run with:

```bash
  server: make && ./ircserv <port> <password>
  client: nc <ip adress> <port>
```
